# 🚀 How to Host Your Mockey Soccer RPG

You have two easy ways to host this game:

## Option 1: GitHub Pages (Recommended)
1.  **Create a new repository** on GitHub (e.g., `mockey-soccer-live`).
2.  **Upload the contents** of this `dist` folder directly to the `main` branch.
3.  Go to **Settings > Pages**.
4.  Under **Build and deployment**, set the source to **Deploy from a branch** and select `main`.
5.  Your game will be live at `https://yourusername.github.io/mockey-soccer-live/`!

## Option 2: Netlify / Vercel (Easiest)
1.  Go to [Netlify](https://www.netlify.com/) or [Vercel](https://vercel.com/).
2.  **Drag and drop** this entire `dist` folder into their upload area.
3.  They will give you a live URL instantly!
